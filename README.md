## 📖 Über dieses Werk
Dieses Repository enthält den vollständigen Quellcode für eine post-nationale, auf Leidminimierung basierende Zivilisation. Es ist kein utopischer Roman, sondern eine technische Spezifikation für **Hardware, Software, Ökonomie und Recht**.

Das System basiert auf einem einzigen Axiom:
> **"Die Legitimität jeder Entscheidung bemisst sich an ihrer Fähigkeit, physisches und psychisches Leid zu minimieren, ohne die Autonomie unter ein kritisches Maß zu senken."**
