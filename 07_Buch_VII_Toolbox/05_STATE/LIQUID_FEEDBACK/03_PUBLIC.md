# 🌐 TOOL: LIQUID-FEEDBACK (KIT)
## Dein Interface zur Macht

So nimmst du aktiv an der Gestaltung des Clusters teil:

### 1. Das "Set and Forget" Prinzip
Wähle in deinem Dashboard für die 12 Fach-Domänen (Anker) Personen aus deinem Cluster aus, deren Urteil du vertraust. Solange du nichts tust, stimmen sie in deinem Namen mit.

### 2. Der "Active Override"
Sobald ein Thema deine persönliche Aufmerksamkeit erregt, kannst du deine Stimme für diesen spezifischen Fall "zurückholen" und selbst votieren. Dein V-Gate hilft dir dabei, sachlich zu bleiben.

### 3. Vorschlagseinreichung (The Proposal Loop)
Jeder Bürger kann einen Vorschlag einreichen. 
- **Phase A (Diskussion):** Der Vorschlag wird im Cluster-Netzwerk debattiert.
- **Phase B (Veredelung):** Die 13. Instanz prüft die formale Logik.
- **Phase C (Abstimmung):** Messung des systemischen Widerstands.

### 4. Benachrichtigungs-Filter
Damit du nicht im "Abstimmungs-Spam" versinkst, filtert das Interface nur die Themen für dich heraus, die eine hohe Leid-Relevanz für dein persönliches Umfeld haben.
