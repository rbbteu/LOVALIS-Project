# TOOL: V-GATE (Entscheidungs-Filter)
[cite_start]**Formula:** V = T ∩ O ∩ E[cite: 168].
Decision is valid only if:
1. **Technical (T):** Is it possible?
2. **Operational (O):** Is it worth it?
3. [cite_start]**Ethical (E):** Is it fair?[cite: 170].

[cite_start]Prevents Sunk Cost Fallacy and forces real-time assessment[cite: 169].
