# Buch I: Die Axiomatik

## Das Kern-Axiom
Das oberste Ziel des Systems ist die Minimierung von Leid (L_min).
Leid ist definiert als der Zustand, in dem ein fühlendes Wesen in seiner physischen oder psychischen Integrität gegen seinen Willen verletzt wird.

## ZVF (Zustands-Validierungs-Funktion)
Jede Entscheidung im System muss folgende Gleichung erfüllen:
V_toe(Op) = (T >= n_min) && (Ö <= E_gen) && (L_inc == 0)
- **T:** Technische Machbarkeit
- **Ö:** Ökologische Verträglichkeit
- **E:** Ethische Unbedenklichkeit
